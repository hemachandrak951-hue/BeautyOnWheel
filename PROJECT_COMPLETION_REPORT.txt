✅ 4-Layer Clean Architecture
   Entity → Repository → Service → Controller

✅ JWT Stateless Authentication
   - Bearer token in Authorization header
   - 24-hour expiration (configurable)
   - Claims extraction: userId, role, email

✅ Role-Based Access Control (RBAC)
   - ROLE_CUSTOMER: View dashboard, services
   - ROLE_STYLIST: Plus schedule management
   - ROLE_ADMIN: Full system access

✅ Amazon/Flipkart-Style Service Catalog
   - Services grouped by category
   - Alphabetically sorted categories
   - Single unified dashboard response

✅ Global Exception Handling
   - 401 Unauthorized (JWT failures)
   - 403 Forbidden (insufficient permissions)
   - 404 Not Found (missing resources)
   - 400 Bad Request (validation errors)
   - Structured error responses

✅ CORS Configuration
   - Enabled for Angular (localhost:4200)
   - Additional origins: 4300, 3000
   - Authorization header explicitly allowed
   - Pre-flight caching: 3600 seconds
